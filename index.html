Cài đặt nhanh (Linux)

Tải script từ download.lt4c.io.vn/install.sh và chạy trực tiếp:

cURL 
curl -fsSL -o /tmp/install_lt4c.sh https://download.lt4c.io.vn/install.sh && sudo bash /tmp/install_lt4c.sh
wget 
wget -qO /tmp/install_lt4c.sh https://download.lt4c.io.vn/install.sh && sudo bash /tmp/install_lt4c.sh
